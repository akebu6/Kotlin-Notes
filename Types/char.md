### Characters
> Each character is just a symbol enclosed in single quotes. 
The Char type represents letters (both uppercase and lowercase), digits, and other symbols.
```js
val lowerCaseLetter: Char = 'a'
val upperCaseLetter: Char = 'Q'
val number: Char = '1'
val space: Char = ' '
val dollar: Char = '$'
```
